# Web Test Automation

The code is made in Cypress and can be executed in any web browser

## Versions:

Cypress package version: 12.12.0
Cypress binary version: 12.12.0
Electron version: 21.0.0
Bundled Node version: 
16.16.0

### Installation

_Below is an example of how you can run this testcase._

1. Clone the repo
   ```sh
   git clone https://github.com/alexbirchler/XXXX
   ```
2. Go to the correct folder in your PC where is allocated the code

3. Run the testcase

    npm run cypress:open
